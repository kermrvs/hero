## Description

Application for heroes management. 
Technologies:
- Mongodb;
- Mongoose;
- Nest;
- TypeScript;
- Jest for test;

## Installation

```bash
$ yarn
```


## Running the app

First step you need to change MONGODB_URI in .env file if you need.

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# integration tests
$ yarn test:integration

# build app
$ yarn build

```
